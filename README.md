# Living Off the Land Attacks - TryHackMe

This repository contains my walkthrough for the TryHackMe room **Living Off the Land Attacks**.

The room focused on how attackers abuse trusted Windows tools to execute commands, download payloads, create persistence, and blend malicious activity with normal admin behaviour.

## Topics Covered

- Living Off the Land attacks
- LOLBins
- PowerShell abuse
- WMIC remote execution
- Certutil download and encoding activity
- Mshta script execution
- Rundll32 DLL execution
- Scheduled Tasks persistence
- SIEM alert review
- Windows command-line analysis

## Summary

In this room, I worked through common Living Off the Land techniques and reviewed alerts related to trusted Windows utilities being abused. The practical part focused on deciding which activity was malicious and which was normal, based on command-line behaviour and context.

Tools covered included PowerShell, WMIC, Certutil, Mshta, Rundll32, and Scheduled Tasks.
